# Smart Laundry Basket

## Introduction

This project is an intelligent laundry basket system with sterilization and dehumidification functions.

The system uses:
- AHT10 humidity sensor
- HX711 weight sensor
- Arduino
- UV sterilization lamp
- Cooling fan
- LED indicators

The purpose of this project is to reduce moisture and bacteria growth inside the laundry basket and improve clothing hygiene.

---

## Features

- Automatic humidity detection
- Automatic fan activation
- UV sterilization
- Weight detection
- LED weight indication

---

## System Architecture

The system architecture is shown in the project report.

Main workflow:
1. Detect humidity
2. Activate fan when humidity exceeds threshold
3. Detect clothing weight
4. Display weight status with LEDs
5. Enable sterilization lamp periodically
